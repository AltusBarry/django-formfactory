Installation
============
#. Install or add ``django-formfactory`` to your python path, e.g.::

    pip install django-formfactory

#. Add ``"formfactory"`` to your ```INSTALLED_APPS``` setting::

    INSTALLED_APPS = [
        # ...
        "formfactory",
    ]
